# Abstract

The Initial Public Offering (IPO) market has emerged as a high-potential investment opportunity, but predicting listing performance remains complex due to market volatility, varying financial boundaries, and unpredictable investor demand. 

This project presents **StockTraQ**, an AI-powered IPO analytics and prediction ecosystem tailored to analyze incoming IPOs through strictly data-driven determinants such as subscription allocation trends (QIB, NII, Retail), company financial fundamentals (Revenue, PAT, ROE, ROCE), and valuation metrics. 

Unlike generalized chatbot platforms that hallucinate financial projections, StockTraQ utilizes an ensemble architecture combining deterministic Machine Learning regressions (Random Forest, Gradient Boosting, Linear Regression) to accurately predict listing gains and generate an actionable Unified IPO Rating (1-10). Additionally, it features a specialized Retrieval-Augmented Generation (RAG) chatbot capable of indexing standard Red Herring Prospectus (RHP) PDF documents to fetch qualitative metrics—such as risk factors and business overviews—transparently. By bridging analytical mathematics, local backtesting algorithms, and conversational AI, StockTraQ provides a cohesive, high-density dashboard empowering investors to execute informed, non-speculative financial decisions.
